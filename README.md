# Thần Số Học Master & Connection Analyst

Ứng dụng Thần Số Học chuyên sâu:  
- Tra cứu chỉ số (Đường Đời, Nội Tâm, Sứ Mệnh, Nhân Cách, Thái Độ, Trưởng Thành, Năm Cá Nhân, Peaks & Challenges...)  
- Phân tích liên kết chỉ số (2 hoặc 3 số, rule engine + AI phân tích hành vi tâm lý)  
- Chatbot AI tư vấn cá nhân hóa (dựa trên chỉ số, hỏi ngược tình huống thực tế)  
- Hỗ trợ tiếng Việt / English  
- Bảo mật cao: API key chỉ nằm ở server Vercel, không lộ trong code client
